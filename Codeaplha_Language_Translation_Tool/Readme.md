# 🌐 AI Language Translation Tool (Python)

## 📌 Overview

This is an AI-powered Language Translation Tool developed in Python as part of my internship at **CodeAlpha (Artificial Intelligence)**. The application provides real-time translation between multiple languages using a simple and interactive GUI built with Tkinter.

It supports multiple languages including **Urdu**, and also includes additional features like text-to-speech and clipboard copy functionality.

## 🚀 Features

* Real-time language translation
* Supports multiple languages (English, Urdu, Arabic, French, Spanish, etc.)
* Simple GUI using Tkinter
* Text-to-Speech (AI voice output)
* Copy translated text to clipboard
* Error handling for smooth user experience

## 🛠️ Technologies Used

* Python
* Tkinter (GUI Framework)
* deep-translator (AI Translation API)
* pyttsx3 (Text-to-Speech Engine)

```bash
git clone https://github.com/your-username/language-translation-tool.git
cd language-translation-tool
```

### 2. Install Dependencies

```bash
pip install deep-translator pyttsx3
```

### 3. Run the Project

```bash
python main.py
```

---

## 🖥️ How to Use

1. Enter text in the input box
2. Select source language
3. Select target language
4. Click **Translate**
5. View translated text
6. Use:

   * **Copy** button to copy text
   * **Speak** button for audio output
